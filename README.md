# AI Support Analytics SQL Project

## Project Overview

This SQL project simulates an AI-powered customer support platform inspired by ChatGPT support systems and AI SaaS products.

The project focuses on:
- AI chatbot analytics
- Customer support analysis
- Ticket management
- Revenue insights
- Escalation trends
- Customer satisfaction tracking

---

## Tech Stack

- SQL (MySQL)
- GitHub

---

## Database Structure

The project contains 7 interconnected relational tables:

1. users
2. ai_models
3. support_tickets
4. chatbot_interactions
5. human_agents
6. ticket_assignments
7. payments

---

## SQL Concepts Used

- SELECT Statements
- WHERE Filtering
- GROUP BY
- Aggregate Functions
- CASE WHEN
- JOIN Operations
- Window Functions
- RANK()
- Business KPI Analysis

---

## Key Business Questions Solved

- Which AI model performs best?
- What is the chatbot escalation rate?
- Which users generate the highest revenue?
- Which issue types are most common?
- How fast are tickets resolved?
- Which agents handle the most tickets?
